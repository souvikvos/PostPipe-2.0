# Create PostPipe CMS

![PostPipe Banner](https://img.shields.io/npm/v/create-postpipe-cms)

A powerful CLI tool to scaffold robust **CMS features** (E-commerce, Blog, Reviews) for Next.js applications, powered by **PostPipe**.

  - `/api/categories`: CRUD categories.

### üìù Blog Module

- **Models**: `Post`, `Comment`.
- **APIs**:
  - `/api/posts`: Full blog post management.
  - `/api/comments`: Commenting system linked to posts.

### ‚≠ê Reviews Module

- **Models**: `Review`.
- **APIs**:
  - `/api/reviews`: Submit and list testimonials/reviews.

## Usage

Run this command in your Next.js project root:

```bash
npx create-postpipe-cms
```

It will prompt you to choose the modules you want to add to your project.
