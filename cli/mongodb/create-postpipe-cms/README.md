# Create PostPipe CMS

CLI to scaffold robust CMS features (E-commerce, Blog, Reviews) for Next.js applications.

## Features

- **Store Module**: Products and Categories API.
- **Blog Module**: Posts and Comments API.
- **Reviews Module**: Testimonials/Reviews API.
- **Database**: Mongoose (MongoDB) support.

## Usage

```bash
npx create-postpipe-cms
```

Select the modules you want to install from the interactive CLI.
